    - Q: 本项目的技术架构是怎样的？
    - A: 本项目前后端分离。
    
        1. 前端使用Vue框架作为骨干，使用Element等UI组件。
        用到的技术栈有: 
        
            - JavaScript
            - Vue
            - Element
            - Axios
        
        2. 后端计划采用 Spring 全家桶，并搭配其他常用技术实现分布式、高可用、微服务、高并发的项目。
        用到的技术栈有: 
        
            - Java
            - SpringBoot
            - MyBatis
            - MySQL
            - Redis
            - Nginx
            - Docker
            - RabbitMQ
            - Elasticsearch
            - Swagger-UI
            - SpringSecurity
            - Druid
